A comprehensive fan website celebrating the Guardians of the Galaxy franchise, featuring character profiles, movie summaries, memorable quotes, and galactic exploration.
This project was developed collaboratively with a group of 4 members.

Project Overview:
This group project was developed for ISTE 140, combining our passion for the Guardians of the Galaxy franchise with web development fundamentals.

Features
Character Profiles
Detailed profiles of main characters from the franchise
Movie Summaries
Comprehensive summaries of films in the Guardians series
Collection of iconic quotes from the movies
Interactive Audio

Technologies Used:
HTML - Structure and content
CSS - Styling and visual design
JavaScript - Interactive audio playback functionality

Purpose
The website serves as an educational and entertainment platform for fans to explore the Guardians of the Galaxy universe, combining our love for franchise's action-packed storytelling.
