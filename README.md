A comprehensive fan website celebrating the Guardians of the Galaxy franchise, featuring character profiles, movie summaries, memorable quotes, and interactive galactic exploration.

## Project Overview
This group project was developed for ISTE 140, combining our passion for the Guardians of the Galaxy franchise with web development fundamentals to create an engaging and interactive fan experience.

## Features:
**Character Profiles**
* Detailed profiles of the main characters from the franchise

**Movie Summaries**
* Comprehensive summaries of films in the Guardians series

**Memorable Quotes**
* Collection of iconic quotes from the movies

**Interactive Audio Elements**
* Click-to-play audio functionality with visual feedback
* Dynamic opacity changes based on playback state

## Technologies Used
* **HTML** - Structure and content organization
* **CSS** - Styling and visual design
* **JavaScript** - Interactive audio playback and user interactions

## Team
This project was developed collaboratively with a group of 4 members.

## Purpose
The website serves as an educational and entertainment platform for fans and newcomers to explore the Guardians of the Galaxy universe, combining our love for franchise's action-packed storytelling and interactive features.
